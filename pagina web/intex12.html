Desarrollo

Bienvenido al proyecto SpeedBots. Este proyecto consiste en un carro a control remoto con un sistema de detección automática utilizando conexiones H y arduinos. Nuestro objetivo es crear un vehículo autónomo capaz de navegar y detectar obstáculos de manera eficiente.

Sobre el Proyecto

¿Por qué elegimos este proyecto?

Este proyecto fue elegido porque se buscó mezclar la programación con arduinos, además de tener una utilidad. En este caso, se puede utilizar para protección o prevención de accidentes, así como para concientizar sobre la seguridad vial y los avances tecnológicos que brindan seguridad.

Materiales

Chasis y Estructuras de Soporte: Optamos por un chasis de plástico resistente para garantizar ligereza y durabilidad. Utilicé estructuras de soporte adicionales para asegurar los motores y otros componentes de forma firme.

Ruedas y ejes: Elegimos ruedas robustas adecuadas para terrenos variados, asegurándome de que los ejes fueran lo suficientemente fuertes para soportar el peso del vehículo y los movimientos bruscos.

Arduino Uno: Decidimos utilizar un solo Arduino Uno para simplificar la configuración y la programación. Este Arduino controla tanto el movimiento del vehículo como el sistema de detección automática.

Motores: Utilizamos motores DC para la propulsión, implementando un puente H para controlar la dirección de giro. Esto permite que se muevan hacia adelante y hacia atrás utilizando el chip L293D.

Sensor ultrasonico: Implementamos un sensor PIR para la detección automática de obstáculos y otros vehículos. Ajustamos los parámetros de sensibilidad para respuestas rápidas y precisas durante las pruebas.

Control Bluetooth: Utilizamos un módulo Bluetooth para el control remoto, asegurando una comunicación estable y eficiente dentro del área de operación deseada.

Baterías: Utilizamos baterías recargables para alimentar el Arduino y los motores, calculando correctamente la capacidad de la batería para mantener la duración operativa deseada.

puente h: Permite invertir la polaridad de la corriente que llega al motor, lo que hace que el motor cambie de dirección. Esto es esencial para un carro a control remoto, ya que permite que el vehículo avance o retroceda según sea necesario.

Conclusión

En conclusión, a lo largo del desarrollo, no solo se adquirieron conocimientos técnicos sobre componentes electrónicos y programación, sino que también se enfrentaron desafíos que requirieron trabajo en equipo y mucha investigación. Este proyecto demuestra cómo la integración de tecnologías simples, como los sensores y el puente H, puede resultar en un sistema autónomo capaz de tomar decisiones en tiempo real, abriendo las puertas a futuras innovaciones en el campo de la robótica móvil.
